# LegalTech Contract Parser

A Django-based LegalTech application for automated contract parsing, clause extraction, and risk analysis.

## Project Objective

The system allows users to upload legal contract PDFs, extract text from documents, identify important clauses, and flag potential risks.

## Tech Stack

- Python
- Django
- PostgreSQL
- PyMuPDF
- spaCy

## Four Week Roadmap

### Week 1
- Django Setup
- PostgreSQL Configuration
- Document Models
- PDF Upload API

### Week 2
- PDF Text Extraction
- Text Cleaning
- Store Extracted Text

### Week 3
- NLP Integration
- Clause Categorization
- Risk Detection

### Week 4
- API Serialization
- Admin Dashboard
- Documentation
